#  Coffee Shop Full Stack Project (ASP.NET + Angular)

This is a full-stack Coffee Shop management system built using:

-  ASP.NET Core Web API
-  Angular 
-  Entity Framework Core
-  JWT Authentication 
-  SQL Server Database

---

## 📁 Project Structure


---

##  Features

###  Authentication
- User Registration
- User Login

###  Products
- Add Product
- Delete Product
- Search Products
- Filter by Price
- Sort Products
- Low stock filter

###  Orders
- Add products to order
- Increase / decrease quantity
- Remove items from order
- Calculate total price
- Create order via API

###  Order Details API
- Get all order details
- Get order by ID
- Get order by product
- Update quantity
- Calculate total per order
